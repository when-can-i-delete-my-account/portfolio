Welcome to the ml5 project! Developing ml5 is not just about developing machine learning software, it is about making machine learning approachable for a broad audience of artists, creative coders, and students. The library provides access to machine learning algorithms and models in the browser, building on top of TensorFlow.js with no other external dependencies.The library is supported by code examples, tutorials, and sample datasets with an emphasis on ethical computing. Bias in data, stereotypical harms, and responsible crowdsourcing are part of the documentation around data collection and usage. We're building friendly machine learning for the web - we're glad you're here! ml5.js is heavily inspired by Processing and p5.js.

HTML (or the Hypertext Markup Language) is the world's biggest programming language for web development. It was developed by Time Berners Lee, the same person who created the Internet. It goes good with CSS(used for style) and JS(used for animation and others).

CSS stands for Cascading stylesheet, and it is created for giving HTML pages some styling. There are two ways it can be implemented; it can be included in the HTML script or have it's own page, with a link to it in the HTML. A programming language more advanced than CSS is JS(javascript). It is used for complex animation and other sorts, such as buttons and sounds.
